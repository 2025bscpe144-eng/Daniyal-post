Semester Project – Machine Learning

Our first semester project was a Machine Learning model. The task was to train a model to predict prices based on different factors.

Steps we followed:

  1. Collected and cleaned data.
  2. Applied a suitable ML algorithm.
  3. Trained and tested the model to evaluate performance.

This project gave me hands-on experience with ML and introduced me to building predictive models.
